Determining Spatial Auto-Correlation with Nominal Variables through Join Count Statistics¶

Matthew Tralka

Objective: Use the join count statistics method to determine the special pattern of Urban and built-up lands in Maryland. Automate methodology using PySAL and seaborn.

Data Source: MODIS - MD extract, multi-polygon shapefile

Join Count Statistics- Methodology for determining the spatial autocorrelation for variables with nominal values. Two values for the variables, 0 or 1. Method measures where these two values touches, determining whether the values are clustered or dispersed.

Null Hypothesis: Perfect random distributions

Check out the official PySAL documentation here and the PySAL Join Count blogpost here

As the mean observed real join count is higher than the expected random join count, there is a difference in real vs expected random distributions.

To determine if this difference is signifiant and reflective of spatial distributions, we can identify our pseudo-p value

In [143]:
join_count.p_sim_bb
Out[143]:
0.001

This value is below the standard significant level of .05.

We fail to support the null hypothesis. The Binary 1 values are significantly not randomly spatialy distributed.

As observed join count is greater than the expected join count, we assume the Binary 1 values clusters in space.

Maryland urban and built-up land cover is signficiantly spatially clustered throughout the state¶
